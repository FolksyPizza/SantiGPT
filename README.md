SantiGPT is an AI model trained for high accuracy & reasoning, built on the GPT-4 Architecture, with support of a context window of up to 32k tokens for long-form reasoning and content generation. 

##Capibilities of SantiGPT/S-GPT
- [x] Text Generation
- [x] Context-aware responses
- [x] Built on the GPT-4 Architecture, with up to a 32k token context window
